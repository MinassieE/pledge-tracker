# NCIC Website Backend

Welcome to Pledge Tracker repository. This backend server is built using Node.js, Express.js, and MongoDB to provide the backend functionality for Pledge Tracker Web application.

## Getting Started

To get started with Pledge Tracker Web app Backend, follow these steps:

1. Clone this repository to your local machine:

2. Navigate to the project directory:

3. Install the project dependencies using npm:

   ```bash
   npm install
   ```

4. Start the server:

   ```bash
   npm run start
   ```

The server will start, and you will see output indicating that it's listening on a specific port. (e.g., `Server started ðŸš€ on port 8001`).
